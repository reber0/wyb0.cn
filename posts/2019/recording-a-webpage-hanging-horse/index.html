0x00 常见网页挂马方式

  • iframe 框架挂马
    简单来说就是加 iframe 标签
  • script 挂马
    通过各种办法加载 js 代码
  • htm 文件挂马 上传 htm 文件，然后用 script 引入
  • js 挂马 上传 js 文件，然后用 script 引入
  • 图片伪装挂马 比较新颖的一种挂马隐蔽方法
  • 等等。。。
